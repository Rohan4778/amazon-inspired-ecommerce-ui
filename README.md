# Amazon Inspired Ecommerce UI

This project is a frontend clone of Amazon homepage built using HTML and CSS.

## Features
- Responsive Navbar
- Search Bar UI
- Hero Section
- Product Cards Layout
- Footer Section

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
